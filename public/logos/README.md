# Fleece AI Logos

Place les logos Fleece AI dans ce dossier :

- `fleece-ai-suite.svg` ou `fleece-ai-suite.png` - Logo principal Fleece AI Suite
- `fleece-ai-consulting.svg` ou `fleece-ai-consulting.png` - Logo Fleece AI Consulting
- `fleece-ai-automatisations.svg` ou `fleece-ai-automatisations.png` - Logo Fleece AI Automatisations
- `fleece-ai-formations.svg` ou `fleece-ai-formations.png` - Logo Fleece AI Formations

## Formats recommandés

- SVG (préféré) pour une qualité optimale
- PNG avec fond transparent en fallback
- Taille recommandée: 400x100px minimum
